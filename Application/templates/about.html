Tweet and Sentiment Analysis

Analyse stats of different Twitter Handles
and
Analyse the sentiments of the tweets too!


This project uses Python Ltural Language Processing in the backend to analyze each tweet
and determine it's Polarity.
The tweet is classified as Positive if Polarity is greater than 0,
neutral if Polarity is equal to 0
and Negative if Polarity is less than 0
